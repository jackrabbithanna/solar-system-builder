# Solar System Builder

Product vision for the project. Current implementation notes and agent guidance live in `docs/`.

Gnome / GTK Python application starting with a Gnome Builder Meson starter project.

This application simulate the orbits of a solar system and play the motion of the stars, planets, and comets according to the equations of general relativity forward and backward in time.

Mass, velocity, and initial position of the solar system bodies should be configurable. 

Multiple solar systems should be able to be created and saved.

The initial solar system should be the solar system we live in.
